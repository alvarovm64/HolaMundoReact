# Proyecto sostenible con React, TypeScript y bootstrap

---

React:Es una biblioteca utilizada para crear interfaces de usuario, especialmente en aplicaciones web interactivas, basada en componentes reutilizables.
Node.js: Entorno de ejecución de JavaScript que permite ejecutar JavaScript fuera del navegador, normalmente en el servidor.
Vite: Es una herramienta de desarrollo frontend que sirve para crear, ejecutar y construir proyectos web rápidamente, proporcionando un servidor de desarrollo rápido y optimización para producción.
